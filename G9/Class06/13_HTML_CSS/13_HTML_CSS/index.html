Formatting Hyperlinks

Styling links

  • Browsers tend to show links in blue with an underline by default, and they will change the color of links that have been visited to help users know which pages they have been to.
  • In CSS, there are two pseudo-classes that allow you to set different styles for links that have and have not yet been visited.

*A CSS pseudo-class is a keyword added to a selector that specifies a special state of the selected element(s).

Styling links (2)

  • :link
    • This allows you to set styles for links that have not yet been visited.
  • :visited
    • This allows you to set styles for links that have been clicked on.
    • *For privacy reasons, the styles that can be modified using this selector are very limited.


a:link {
	color: deeppink;
	text-decoration: none;
}
a:visited {
	color: black;
}

Styling links (3)

  • There are three pseudo-classes that allow you to change the appearance of elements when a user is interacting with them.

Styling links (4)

  • :hover
    • This is applied when a user hovers over an element with the mouse.
  • :active
    • This is applied when an element is being activated by a user; for example, when a button is being pressed or a link being clicked.
  • :focus
    • This is applied when an element has focus. Focus occurs when a browser discovers that you are ready to interact with an element on the page.


a {
	text-decoration: none;
}

a:link {
	color: #265301;
}

a:visited {
	color: #437A16;
}

a:focus {
	background-color: #BAE498;
}

a:hover {    
	background-color: #CDFEAA;
}

a:active {
	background-color: #265301;
	color: #CDFEAA;
}

*When pseudo-classes are used, they should appear in this order: :link, :visited, :hover, :focus, :active.
