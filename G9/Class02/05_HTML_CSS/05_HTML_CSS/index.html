Adding images

Adding images

  • <img />
  • Required Attributes: src and alt
  • More info: https://developer.mozilla.org/en-US/docs/Web/HTML/Element/img

Example


<img src="img/imagename.png" alt="image description" />

src & alt

  • src - The image URL. Mandatory for the <img /> element.
  • alt - Defines an alternative text description of the image

Note: Browsers do not always display images. There are a number of situations in which a browser might not display images. Always use the alt attribute to provide an alternative for all users

Width & Height

  • width - specifies the width of the image in pixels
  • height - specifies the height of the image in pixels

PRO TIP: Use just one of them to keep the aspect-ratio / proportions of the image.

*If not specified the image is displayed in its original dimensions

Create image links

  • You can easily create link around an image by wrapping it inside Hyperlink anchor tag


<a href="link-here"/><img src="img/image.png" alt="image description"/></a>

Image formats

Data Compression

  • Images are typically large date size files and thus not very friendly for web use without some kind of compression
  • Data Compression is a technique in which the size of data is reduced without loss of information

Data Compression (2)

  • Lossy compression and Lossless compression are the two categories of data compression methods
  • The main difference between the two compression techniques is that, lossy compression technique does not restore the data in its original form after decompression and lossless compression does

Image formats

  • JPEG
  • The JPEG image format is currently the most widely used lossy compression format for still images
  • It's particulary useful for photographs
  • Whenever you have many different colors in a picture you should use a JPEG
  • Doesn't support alpha transparency
  • File extension(s): .jpg, .jpeg, .jpe, .jif, .jfif

Image formats (2)

  • PNG
  • The PNG image format uses lossless compression to provide more efficient compression, and supports higher color depths than GIF, as well as full alpha transparency support
  • PNG pictures are a good choice for line drawings and other textual or iconic graphics because of the sharp contrasts between adjacent pixels
  • File extension(s): .png

Image formats (3)

  • APNG
  • APNG is a file format first introduced by Mozilla which extends the PNG standard to add support for animated images
  • Conceptually similar to the animated GIF format which has been in use for decades, APNG is more capable in that it supports a variety of color depths, whereas animated GIF supports only 8-bit indexed color
  • File extension(s): .apng

Image formats (4)

  • GIF
  • GIF was one of the first two graphics formats supported by HTML, along with XBM
  • Each pixel in a GIF is represented by a single 8-bit value serving as an index into a palette of 24-bit colors (8 bits each of red, green, and blue)
  • GIF supports simple animation as well as transparency in the pictures
  • File extension(s): .gif

Image formats (5)

  • GIF
  • GIF has been extremely popular for decades, due to its simplicity and compatibility
  • Its animation support caused a resurgence in its popularity in the social media era, when animated GIFs began to be widely used for short "videos", memes, and other simple animation sequences
  • Typically, modern content should use PNG for lossless and indexed still images, and should consider using APNG for lossless animation sequences

Image formats (6)

  • WebP image
  • Lossy WebP images average 25–35% smaller than JPEG images of visually similar compression levels
  • Lossless WebP images are typically 26% smaller than the same images in PNG format
  • WebP also supports animation
  • WebP now has broad support in the latest versions of major web browsers, although it does not have deep historical support
  • File extension(s): .webp

Image formats (7)

  • SVG (Scalable Vector Graphics)
  • SVG is a vector graphics format that specifies the contents of an image as a set of drawing commands that create shapes, lines, apply colors, filters, and so forth
  • SVG files are ideal for diagrams, icons, and other images which can be accurately drawn at any size
  • As such, SVG is popular for user interface elements in modern Web design
  • File extension(s): .svg

Image formats (8)

  • SVG (Scalable Vector Graphics)
  • SVG files are text files containing source code that, when interpreted, draws the desired image
  • SVG can be used in web content in two ways:
    1. You can directly write the <svg> element within the HTML, containing SVG elements to draw the image
    2. You can display an SVG image using the <img /> element same as with all other image formats

Image optimization

Image optimization

  • Image optimization is about reducing the file size of your images as much as possible without sacrificing quality so that your page load times remain low.
  • It’s also about image SEO. That is, getting your product images and decorative images to rank higher on Google and other image search engines.

Name your images descriptively and in plain language

  • When it comes to image SEO, it's important to use relevant keywords to help your webpage rank on search engines
  • Creating descriptive, keyword-rich file names is crucial for image optimization
  • Search engines not only crawl the text on your webpage, they also crawl your image file names

Optimize your alt attributes carefully

  • Alt attributes are the text alternative to images when a browser can't properly render them
  • They’re also used for web accessibility
  • Adding appropriate alt attributes that include relevant keywords to the images on your website can help you rank better in the search engines
  • Setting this attribute to an empty string (alt="") indicates that the image is not a key part of the content (it’s decoration or a tracking pixel), and that non-visual browsers may omit it from rendering

Image dimensions

  • The images you use on your website should be saved at the same width and height that you want them to appear on the page wherever possible
  • When cropping images it is important not to lose valuable information
  • Be careful providing larger images to your visitors, as they will increase your page load time
  • Instead, you can make a smaller image (thumbnail) and provide the option to view a larger image in a pop-up window or on a separate webpage

Choose the right file format

  • Choose the format that will provide the best quality for the smallest file size
  • Learn more about the usage of different image file formats to make informed decisions

Image Map

Image Map

  • Image maps are a bit more complex than simple images
  • To create image map, you can use <map> tag with specifying one or more <area /> tags combined with
    <img /> tag which is linked to the <map>
  • MDN page for image maps: https://developer.mozilla.org/en-US/docs/Web/HTML/Element/map

Image Map


<img src="https://placehold.it/350x150" 
alt="350 x 150 pic" usemap="#primary" />

<map name="primary">
	<area shape="circle" coords="75,75,75" href="left.html" />
	<area shape="circle" coords="275,75,75" href="right.html" />
</map>
